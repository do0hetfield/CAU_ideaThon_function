# Figma MCP Connection Guide

To connect your Figma designs to this project and enable seamless design-to-code workflow, follow these steps:

## 1. Setup Figma MCP
If you are using **Cursor** or any IDE that supports MCP (Model Context Protocol), ensure you have the Figma MCP server configured.

- **Figma Personal Access Token**: You will need a PAT from Figma (Settings -> Tokens).
- **Team ID**: Your Figma Team ID.

## 2. Link Figma File
You can provide the Figma URL to the AI assistant to reference specific wireframes.

Example prompt:
> "Check the wireframes at [FIGMA_URL] and implement the Login screen following the design."

## 3. Cursor Rules (Optional)
Add a rule to `.cursor/rules/figma.md` to always check Figma designs before implementing UI components.

```markdown
# Figma Design Rule
- Always check the linked Figma file for specific dimensions, colors, and spacing.
- Use the design tokens defined in `src/index.css`.
```
